# Student Performance Predictor

A Machine Learning web application that predicts student performance based on study hours, attendance, and sleep hours.

---

## Features
- Predicts student score using Machine Learning
- Interactive Flask web application
- Simple and responsive UI

---

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML/CSS

---

## Project Structure

student-performance-predictor/
│
├── app.py
├── model.py
├── student_data.csv
│
└── templates/
    └── index.html

---

## How to Run

### 1. Install dependencies

```bash
pip install pandas scikit-learn flask
```

### 2. Run the project

```bash
python app.py
```

### 3. Open browser

```text
http://127.0.0.1:5000
```
